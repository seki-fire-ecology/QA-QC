# QA-QC
Code for QA/QC of SEKI fire effects data exported from FFI.

The R script searches for errors in the exported FFI data and flags the error for a human to check. The output of the script is an error log for each protocol including the errors found for each plot and read with comments so that the error can be found in the database by a human to examine and correct.

Acknowledgements: This code was adapted from other NPS Fire Ecology programs, specifically Eva Deegan from Saguaro National Park and Alexandra Lalor from Shenandoah National Park. Sequoia Fire Fellows Jacob Davison and Sophie Dua adapted the code for Sequoia & Kings Canyon National Parks.
